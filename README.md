# Number Guessing Casino Game in C

A console-based Number Guessing Casino Game developed in C programming language. The game allows players to place bets, guess randomly generated numbers, earn rewards, and manage a virtual wallet.

## Features

* Random Number Generation
* Multiple Difficulty Levels (Easy, Medium, Hard)
* Betting System
* Wallet Management
* Hint System
* Player Ranking System
* Game Statistics Tracking
* File Handling for Data Storage
* Game History Recording

## Technologies Used

* C Programming Language
* Standard Libraries:

  * stdio.h()
  * stdio.h()
  * time.h()
  * string.h()

## How to Run

1. Compile the source code:
   gcc number_guessing_game.c -o game

2. Run the executable:
   ./game

## Project Objectives

* Develop an interactive number guessing game.
* Implement betting and reward mechanisms.
* Demonstrate file handling in C.
* Apply random number generation techniques.
* Track player statistics and game history.
